# IPTV Player

A modern IPTV player built with Next.js 15, Supports M3U/M3U8 playlists, HLS streaming, Favorites. Custom sources. And authentication.

## 📖 Documentation

**[User Manual (Arabic)](./manual/user-guide.html)** - Complete guide in Arabic

## Features

- 📺 HLS/M3U8 streaming support
- ⭐ Favorites system with local storage
- ➕ Add custom M3U sources with username/password
- 🔍 Search and filter channels
- 📂 Category-based organization
- 🗑️ Hide unwanted channels
- 📱 RTL Arabic interface
- 💾 Persistent settings in browser

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- HLS.js
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Adding Sources

1. Select a source from dropdown or add custom
2. Enter M3U URL
3. Add username/password if required
4. Click "Load" to fetch channels
